How We Calculate Your Price

Pricing Structure:

Total Cost = Base Fee + Transport Cost + Package Size Charge + Documentation Fee

1. Base Fee

Fixed administrative and handling fee based on transport mode:

Transport Mode Base Fee
Air Freight R 2,500
Sea Freight R 1,800
International Courier R 800

2. Transport Cost by Destination

Varies by destination continent and transport mode:

Destination Air Sea Courier
Europe R 45,000 R 25,000 R 3,500
North America R 52,000 R 28,000 R 4,200
Asia R 38,000 R 22,000 R 3,200
Australia R 55,000 R 30,000 R 4,500

3. Package Size Charges

Based on your selected package size and transport mode:

Air Freight Rates:

Package Size Weight Range Rate

4. Documentation Fee

Covers export documentation processing and customs clearance:

  • Air Freight: R 1,500
  • Sea Freight: R 1,200
  • International Courier: R 0 (included in base)
Example Calculation (Air to Europe, Medium Package):

Base Fee: R 2,500

Transport Cost: R 45,000

Package Size Charge: R 2,100

Documentation Fee: R 1,500

Total = R 2,500 + R 45,000 + R 2,100 + R 1,500 = R 51,100
